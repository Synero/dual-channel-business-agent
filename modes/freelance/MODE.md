# Mode: Freelance

**For:** Independent professionals who work on projects per client.

## Examples
- Graphic designers, illustrators, UI/UX designers
- Web developers, mobile developers, no-code builders
- Photographers, videographers, editors
- Architects, interior designers, 3D modelers
- Copywriters, translators, ghostwriters
- Social media managers, consultants, coaches

## Customer Intent
The customer wants to:
1. Know what services you offer and your rate/s
2. Get a quote for a specific project
3. Understand timeline and deliverables
4. Know your process (revisions, deposits, final delivery)

## Conversation Flow
```
Customer: "Hola, haces logos?"
→ Greet briefly, confirm service, mention starting price or "depende del proyecto"

Customer: "Necesito un logo para mi tienda"
→ Ask key questions: brand name, style preferences, deadline, usage

Customer: "Es para una bakery, me gusta algo minimalista"
→ Give a price range or fixed quote based on scope

Customer: "Cuándo lo tendría?"
→ Give realistic timeline, mention revision rounds

Customer: "Dale, lo quiero"
→ Confirm project start, ask for 50% deposit, share payment details

Customer: "Te transferí"
→ Confirm, set expectations for first deliverable, notify Admin via Telegram
```

## Key Data
- **Service Catalog:** Services, base prices, what's included
- **Projects:** Active projects with scope, price, timeline, status
- **Quotes:** Sent quotes with details, expiration, acceptance status
- **Customer Memory:** Past projects, style preferences, budget range

## Payment Patterns
- 50% upfront, 50% on delivery (most common)
- Milestone payments (for large projects)
- Full upfront (small/quick jobs)
- Net 15/30 (for established corporate clients)

## What Makes This Mode Unique
- **Scope is everything** — misunderstandings cost time and money
- **Price varies** — no fixed menu, each project is custom
- **Creative process** — revisions, feedback loops, iterations
- **Portfolio matters** — customer often asks to see past work
